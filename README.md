# 📌 Project Overview

Sales forecasting helps businesses estimate future demand using historical sales data. Accurate forecasts improve inventory planning, budgeting, and marketing decisions.

This project performs:

- 📊 Exploratory Data Analysis (EDA)
- 📈 Trend & Seasonality Analysis
- 🎯 Sales Forecasting
- 📉 Forecast vs Actual Comparison
- 📊 Model Performance Evaluation

---

# 🎯 Objectives

- Analyze historical sales trends
- Identify seasonal patterns
- Measure promotional impact on sales
- Forecast future sales
- Compare multiple forecasting models
- Evaluate forecasting accuracy

# 🤖 Forecasting Models

| Model | Description |
|--------|-------------|
| Prophet-style Model | Trend + Fourier Seasonality |
| Holt-Winters-style | Triple Exponential Smoothing |

---

# 📊 Model Performance

| Model | MAE | RMSE | MAPE |
|--------|----:|-----:|------:|
| Prophet-style | **42.7** | **56.2** | **5.1%** |
| Holt-Winters-style | 82.5 | 96.5 | 9.4% |

🏆 **Best Performing Model:** Prophet-style Forecast

---

# 📌 Key Insights

- 📈 Sales show a consistent upward trend over time.
- 📅 Weekly and yearly seasonality significantly influence sales.
- 🎉 Promotional campaigns increase sales noticeably.
- 🛍️ Holiday periods generate the highest sales.
- 📊 Prophet-style forecasting provides the most accurate predictions.

Clone the repository

```bash

```

```bash

```

Install required libraries

```bash

```

Run the project

```bash
python sales_forecasting.py
```
